# OSM_tourismus_analyse

Dieses Repository enthält die Datengrundlage des Projektes "Räumliche Strukturen im Tourismus: Identifikation und
Visualisierung touristischer Cluster in urbanen und ländlichen
Räumen auf Basis von OpenStreetMap", das im Rahmen des Moduls Introduction to Digital Humanities entstanden ist, sowie die Auswertung dieser.
